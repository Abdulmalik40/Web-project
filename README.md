## Getting Started

1. Navigate to the frontend directory:
   ```bash
   cd frontend
   ```

2. Start a local server:
   ```bash
   python -m http.server 8000
   ```

3. Open your browser and visit:
   ```
   http://localhost:8000/pages/
   ```
